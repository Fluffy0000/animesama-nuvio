# Nuvio Repo

Providers Nuvio : French-Manga + Anime-Sama.

## Installer
Nuvio -> Reglages -> Plugins -> Ajouter depot -> coller l'URL RAW du manifest :
https://raw.githubusercontent.com/<user>/<repo>/refs/heads/main/manifest.json

Pour ajouter un provider : deposer le .js dans providers/ et ajouter une entree dans manifest.json (tableau "scrapers"). Bump la version a chaque changement.
