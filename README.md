## 📥 Installation dans Nuvio

C'est **hyper simple**, 30 secondes ⏱️

1. Ouvre **Nuvio**
2. Va dans **Réglages ⚙️ → Plugins** (ou *Extensions / Sources*)
3. Clique sur **Ajouter un dépôt** (Add repository)
4. Colle ce lien 👇

```
https://raw.githubusercontent.com/Fluffy0000/animesama-nuvio/refs/heads/main/manifest.json
```

5. Valide ✅ → **🐍 Anime-Sama** apparaît dans ta liste
6. Ouvre un anime, un film… et régale-toi 🍿

> 💡 **Astuce** : si tu avais déjà une ancienne version, **supprime puis re-ajoute** le dépôt pour forcer la mise à jour.

## 💡 Petits conseils

- 🥇 La **première source** proposée est la meilleure qualité — clique dessus en priorité.
- 🐢 Ça bufferise un peu ? Essaie une source **Sibnet** (fichier direct, souvent le plus fluide) ou une qualité **480p**.
- 🔄 Un anime ne remonte rien ? Le contenu n'existe peut-être pas (encore) sur Anime-Sama — réessaie plus tard.
